# Redux まわり、まとめ

- reducer 周り
  - 容易に自動テスト可能
- middleware
  - interface injection で外界・副作用をモック
- selector
  - reselect で細かくちぎって断面を増やす
- usecase
  - redux-mock-store
- connect はシンプルに保ち、自動テストしない

RN を 0.59 にして Hooks 使い始めると変わってくるかも...？

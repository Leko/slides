## テキスト、まとめ

- Nested Textは基本楽使いこむとすると複雑
- 座標はnestしてる親が基準
- nested textでonLayoutは動かない
- .measureも子テキストに対して効かない（親の座標が取れる）
- 単語選択の単位＝`<Text />`ごと

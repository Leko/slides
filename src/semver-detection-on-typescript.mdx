## 整形して、比べて、計算する

1. .d.ts から Public API の JSON を生成する（ここまでの話）
   - 型と関係のない情報を取り除く（引数名とか）
1. JSON Patch（RFC6902）で JSON を比較
1. 出てきた差分を分類する
1. 破壊的変更に相当する差分があるかないかチェック
